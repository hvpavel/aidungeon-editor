World Info

Here you can add world info entries (comma separated keys and an entry) to shape what the AI knows about the world and hidden info the player doesn't see.

An entry is included in what's fed into the AI if one of the keys is detected (case doesn't matter) in the last output or input.

  • —
    Download world info file ( .json ) to your local drive. You can edit it just like a simple text file.
  • —
    Upload* world info file from your local drive to this adventure. Current world info will be replaced!
  • —
    Save your current world info with latest changes. You added one or a few new entries - press save.
  • —
    Create* new card, field in card or folder.
  • —
    Delete* selected card from this world info. AI will forget about this entry with some time.
  • —
    Drag and drop* card to new place or folder.
  • —
    Duplicate* current card to active folder. Make quick variations with some changes.
  • —
    Clear* all card’s fields to default type. Custom fields will be deleted.
  • —
    You need to save current world info to apply this changes.
